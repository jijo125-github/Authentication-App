## Authentication App

**The main tasks to be accomplished**

* [ ] Register a new account.
* [ ] Login.
* [ ] Login with one of the following services - Google / Facebook.
* [ ] SignOut.
* [ ] See profile details
* [ ] Edit details including photo, name, bio, phone, email & password.
* [ ] Upload a new photo
